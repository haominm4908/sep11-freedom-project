# SEP11 Freedom Project
by **Felix Mo**

## Context
I am a student at [HSTAT](https://www.hstat.org/) in the [Software Engineering Program](https://hstatsep.github.io/). The "Freedom Project" for SEP11 is a year-long project all about making something using JavaScript along with a third-party JS tool.

For my project, I chose to independently study **gDevelop** in order to help me make my **game**. I chose gDevelop since it had the the easy applicable 3d physics I couldimplement in my game easily. The whole game is a 3D platformer where you as the character jump ontop of platforms and go up your way to the top. The game is simple, and isn't meant to be overcomplicated. Just climb. I mainly implemented a respawn mechanic incase you fall, and the rest is pure skill in whether you can get to the top.

##### PROS
Social

* Players can challenge themselves to do it in a timed fashion with their own timer
##### CONS
Social
* The game has no meaningful impact outside of solving temporary boredom
* Shows the bare minimum in some 3D platformers
